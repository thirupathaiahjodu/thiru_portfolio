

# Developer Portfolio Website

A premium dark-theme portfolio with glassmorphism design, neon gradients, and smooth animations — inspired by high-end SaaS landing pages.

## Visual Identity
- **Dark navy/black background** with subtle gradient overlays
- **Neon gradient text** (cyan → purple → blue) for the name/headings
- **Glassmorphism cards** with backdrop blur, semi-transparent borders
- **Glowing buttons** with gradient backgrounds and soft box-shadows
- **Floating particle background** using animated dots/circles
- **Smooth scroll-triggered fade-in animations** on all sections

## Page Sections

### 1. Navigation Bar
- Fixed top nav with glass effect background
- Links: Home, About, Projects, Skills, Certifications, Contact
- **"Download Resume"** gradient button on the right
- Mobile hamburger menu

### 2. Hero Section (SaaS-style)
- Large animated gradient name text with shimmer effect
- Role subtitle: "Backend Developer / Full Stack Developer"
- Tagline inside a glassmorphism card
- Social icons row (GitHub, LinkedIn, Instagram) with hover glow
- Floating particles animating in the background

### 3. About Section
- Glass card layout with a brief bio
- Clean typography, minimal text, high visual impact

### 4. Projects Section
- Grid of modern project cards with glassmorphism styling
- Hover effects: card lift, border glow, subtle scale
- Each card shows project title, description, tech used, and links
- Placeholder projects to customize later

### 5. Skills Section
- Tech stack displayed as glowing chips/badges
- Python, Django, Django REST Framework, React, JavaScript, MySQL, Git
- Subtle pulse/glow animation on hover

### 6. Certifications Section
- Clean card layout for listing certifications
- Placeholder entries to fill in later

### 7. Contact Section
- Floating glassmorphism form (name, email, message)
- Gradient submit button with glow effect
- Social links repeated at bottom

## Responsive Design
- Mobile-first layout, fully responsive across all breakpoints
- Stacked sections on mobile, grid layouts on desktop

## Technical Approach
- All frontend, no backend needed
- Smooth scroll between sections (single-page app)
- CSS animations and Tailwind utilities for all effects
- Particle background using animated CSS elements
